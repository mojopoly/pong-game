# Pong Game

A basic pong game using SVGs.

## Setup

**Install dependencies:**

`> npm i`

**Run locally with Webpack Dev Server:**

`> npm start`

**Build for production:**

`> npm run build`

## Keys

**Player 1:**
* a: up
* z: down
* w: up 4x
* x: down 4x
**Player 2:**
* ▲ : up
* ▼: down
*p: up 4x
*l: : down 4x
